CSS Display Property

Explanation

Summary

  • Two types of elements in HTML - Block and inline elements
  • Block level elements occupies entire width of the screen and it doesn't allows other elements (both inline/block) to start next to it in the same line.
  • Inline elements only occupies based on content it occupies
  • But, we can override the above display types. i.e. we can make block level elements behave like inline and vice versa.

Display Types

  • Block: When any element is set as 'Block', it behaves like a block element where no other element can start next to it in the same line. Width and Height can be controlled.
  • Inline: When any element is set as 'Inline', it behaves like inline element where it doesn't occupies the entire width of the screen. Height and Width cannot be controlled. Height and Width is dictated by its content.
  • Inline-Block: When set with this, effectively we can make any element's (especially inline elements)
    • width and height can be controlled and other elements can appear next to it in the same line.
